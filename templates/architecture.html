04

Key Capabilities

What makes F1InsightAI different from a basic Text-to-SQL system

🎯

Schema-Aware RAG

Retrieves only relevant tables instead of sending the entire 100+ column schema

🔄

Auto Error Recovery

Failed queries are fed back to the LLM with error context for automatic correction (up to 2 retries)

🪞

Self-Reflection

The reflect node validates results — checking for empty data, errors, and anomalies before answering

💬

Multi-Turn Context

Last 20 messages are injected for pronoun resolution and follow-up awareness

📈

Live RAG Metrics

MRR, Recall@K, Context Relevance, and Faithfulness computed per query

🏎️

F1 Domain Knowledge

Team name changes, race name history, circuit mappings, and continent-based filtering baked into prompts
